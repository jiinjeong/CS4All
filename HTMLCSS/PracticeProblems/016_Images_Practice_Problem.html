Images Practice Problem

  • Create an HTML file and link it to a CSS file.
  • Download an image from the Internet to use as a banner, and embed it in your webpage. In the CSS document, give it a width of 100% and a height of 200 pixels.
  • Copy the image address of another image, and embed it in your webpage. Using an HTML attribute, give it a width of 400 pixels.
  • Add one more image using a method of your choice, and give it an alternate text.
    • Then, temporarily mess up the filepath or image address to see the image replaced by the alternate text.
