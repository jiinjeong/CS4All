Introduction to Specificity

What happens if two or more of our CSS rules contradict each other? The more specific rule will be applied.

  • Inherited styles have a specificity of 0.

  • Styles targeting a type of HTML element (a certain HTML tag) have a specificity of 1.

    • This is a paragraph. Although it has inherited a color of red from the list, a color of blue is applied directly to paragraphs.

  • Styles targeting classes have a specificity of 10.

    • This is a paragraph with a class of "my-class". Although a color of blue is applied to paragraphs (and this paragraph has inherited a color of red from the list), a color of green is applied to the class "my-class."

  • Styles targeting IDs have a specificity of 100.

    • This is a paragraph with an ID of "my-ID" and a class of "my-class". Although a color of green is applied to the class "my-class" (and a color of blue is applied to paragraphs, and this paragraph has inherited a color of red from the list), a color of purple is applied to the ID "my-ID".
