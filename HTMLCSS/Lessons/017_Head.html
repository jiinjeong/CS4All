The HTML Head and Metadata

The head of an HTML document contains information about the webpage, called metadata.

The metadata includes:

  • the character encoding
  • the title of the webpage
  • a description of the webpage (Optional)
  • key words (occasionally used by search engines) (Optional)

If you have not already, open this webpage in a browser. Notice the title displayed in the tab at the top.

If your webpage comes up in a Google search, the title and description may be displayed.
