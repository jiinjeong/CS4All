Images

Embedding Images

To embed an image in a webpage, we use the image tag. The source attribute specifies the location of the image.

The location of the image can be a relative file path...

or the image address of an image from the Internet.

(To get the image address, right click on an image from the Internet and select "Copy image address".)

Resizing Images with Pixels

To control the size of an image, we can give it a width and/or height attribute in the HTML code.

However, it is more common to give it a width and/or height property in the CSS code.

Notice that, when we specify either the width or the height, the image will retain its original proportions. But if we specify both, it may be stretched or squished.

Alternate Text

If the computer cannot find or load an image, it displays this instead:

But what if the image contains very important information? Is there a way to provide that information to the user, even if the image cannot be loaded?

Yes! We can give an image an alternate text. If the computer cannot load the image, it will display the alternate text instead.

If the computer is able to load the image, it will not display the alternate text.
