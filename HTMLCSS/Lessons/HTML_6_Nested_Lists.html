A nested list is a list inside of another list.

A list item can be a nested list. For example:

    1. Point 1-1
    2. Point 1-2

    1. Point 2-1
    2. Point 2-2

The nested list can be the same type as the list that contains it, or a different type.

    1. This is an ordered list...
    2. nested inside an unordered list.

Nested lists can be used to create points and subpoints.

  • This is the point.
    • This is a subpoint.
    • This is another subpoint.
  • Another point.
    • This subpoint has subpoints:
      • Sub-subpoint 1
      • Sub-subpoint 2
    • Another subpoint.
  • Did you notice how nested, unordered lists get stylized bullet points?
    • Isn't that nifty?

Also notice how a point and its list of subpoints are all contained within the same list item.
