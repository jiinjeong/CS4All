Inheritance and Division

Inheritance

If we apply a CSS rule to an element, the rule will also be applied to any elements nested within that element.

For example, this blockquote has inherited a font of Arial from the body.
Also, this "strong" tag has inherited a color of blue from the "blockquote" tag.

Division

Notice how "div" tags divide this code into logical sections.

Independently, a `div` tag has no effect on the webpage.
However, when used together with IDs and classes, a "div" tag is useful for applying different styles to different sections of your webpage.

For example, each section of this webpage has a different background color.
