IDs and Classes

IDs

Any HTML element can be given an ID attribute. Then, we can apply unique styles to it by targing the ID in our CSS code.

Warning: Every ID should be unique! Only one element should have a given ID, and a given element should have only one ID, if any.

In CSS, a hashtag, "#", at the start of the selector means that the target is an ID.

Classes

Any HTML element can be given an class attribute. Usually,there are multiple elements in each class. We can apply unique styles to the elements by targeting the class in our CSS code.

Note: Not only can multiple elements belong to a given class, but a given element can belong to multiple classes. In the value of the "class" attribute, separate different class names with a space.

In CSS, a period, ".", at the start of the selector means that the target is a class.
