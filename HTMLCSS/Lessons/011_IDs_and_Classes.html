IDs and Classes

IDs

Any HTML element can be given an ID attribute. Then, we can apply unique styles to it by targing the ID in our CSS code.

Note: Every ID should be unique! Only one element can have a given ID.

In CSS, a hashtag, "#", at the start of the selector means that the target is an ID.

Classes

Any HTML element can be given an class attribute. Usually,there are many elements in each class. We can apply unique styles to the elements by targeting the class in our CSS code.

In CSS, a period, ".", at the start of the selector means that the target is a class.
