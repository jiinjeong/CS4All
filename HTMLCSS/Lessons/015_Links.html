Links

We can add four types of links to our webpages:

  • Internal Links (link to a different page of the same website)
    • Use a relative file path.
    • For example: Visit this sample page
      • Note: For this link to work, please download "sample.html" or create a "sample.html", and, if necessary, adjust the relative file path.
  • External Links (link to a different website)
    • Use a URL, beginning with "http://".
    • For example: Google
  • Download Links (link to download a file)
    • Use a relative file path.
    • For example: Download this sample PDF
      • Note: For this link to work, please download "sample.pdf" or create a "sample.pdf", and, if necessary, adjust the relative file path.
  • Internal Links (link to a different section of the webpage)
    • Use an ID, beginning with "#".
    • For example: Go to the bottom of this webpage

If you would like a link to open in a new tab, give it the target attribute with a value of "_blank".

  • For example:
    • Visit this sample page
    • Google
    • Download this sample PDF
    • Go to the bottom of this webpage
